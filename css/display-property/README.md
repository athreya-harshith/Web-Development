Display Property

values : {inline,block,inline-block}

1. block
    * newline
    * margin
    * padding
    * width and height
2. inline
    * no new line
    * only vertical margin
    * padding
    * no width and height
3. inline-block
    * no new line (only taken if needed)
    * margin
    * padding
    * width and height
    * divs with this property come in the same line if there is required amount of space
